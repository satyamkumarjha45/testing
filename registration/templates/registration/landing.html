Workshop Overview:

Goal: This workshop equips participants with foundational skills in R programming, data manipulation, and statistical analysis, setting them up for data science applications.

1. Introduction to Data Science and R
  • • Overview of Data Science: Explore the essentials—data, analysis, and insights—and the role of R in data science.
  • • Getting Started with R: Install R and RStudio, and learn the interface (console, script editor, plots).
  • • Hands-On: Basic commands and variable assignments.
2. Basics of R Programming
  • • Data Types & Structures: Vectors, matrices, lists, and data frames.
  • • Operations & Functions: Arithmetic, logic, and creating custom functions.
  • • Data Import/Export: Load from CSV and save data.
  • • Hands-On: Work with small datasets—subsetting and data manipulation.
3. Data Manipulation & Visualization
  • • Data Wrangling with dplyr: Filter, mutate, summarize, and group operations.
  • • Visualization with ggplot2: Build histograms, scatter plots, and customize visuals.
  • • Hands-On: Clean data with dplyr and visualize distributions.
4. Introduction to Descriptive Statistics
  • • Descriptive Measures: Mean, median, variance, and data distribution.
  • • Exploratory Data Analysis (EDA): Use summary() and visualize distributions.
  • • Hands-On: Calculate and interpret basic statistics.
5. Introduction to Inferential Statistics
  • • Hypothesis Testing: Null vs. alternative hypotheses, p-values, significance.
  • • Common Tests: T-tests and chi-square tests.
  • • Hands-On: Run tests in R and interpret outputs.
